# NookSpace

NookSpace; Electron, React ve TypeScript ile geliştirilmiş masaüstü bir AI agent çalışma alanıdır.
Agent'ların araç kullanımı, dosya işlemleri, komut çalıştırma ve harici servis entegrasyonlarını kontrollü bir ortamda yönetir.

## Neden NookSpace?

- Dosya/komut işlemlerinde güvenli varsayılan sınırlar
- Sohbet, iz kayıtları, araç görünürlüğü ve oturum yönetimi odaklı masaüstü deneyimi
- DOCX, PDF, PPTX, XLSX gibi döküman akışları için yerleşik skill sistemi
- Anthropic ve OpenAI uyumlu çoklu model sağlayıcı desteği
- MCP tabanlı genişletilebilir entegrasyon katmanı

## Hızlı Başlangıç

```bash
npm install
npm run rebuild
npm run dev
```

Build alma:

```bash
npm run build
```

## Temel Scriptler

- `npm run dev`: geliştirme modunu başlatır
- `npm run build`: üretim build ve Electron paketlerini oluşturur
- `npm run test`: Vitest testlerini çalıştırır
- `npm run lint`: ESLint kontrolü yapar
- `npm run format`: renderer kodlarını Prettier ile formatlar
