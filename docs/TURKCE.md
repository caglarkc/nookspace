# NookSpace Dokumantasyon (Turkce)

Bu sayfa NookSpace projesinin Turkce genel dokumantasyon ozetidir.

## NookSpace Nedir?

NookSpace, Electron + React + TypeScript ile gelistirilmis masaustu AI ajan calisma alanidir.
Amaci, ajan ile dosya islemleri, komut calistirma ve entegrasyon sureclerini kontrollu bir ortamda yonetmektir.

## Temel Ozellikler

- Guvenli calisma klasoru sinirlari
- Arac cagrilari ve iz paneli ile masaustu odakli deneyim
- Dahili beceri (skills) sistemi: DOCX, PDF, PPTX, XLSX
- Anthropic ve OpenAI-uyumlu saglayicilarla model destegi
- MCP tabanli entegrasyon altyapisi

## Kurulum (Kisa)

```bash
npm install
npm run rebuild
npm run dev
```

Uretim paketi almak icin:

```bash
npm run build
```
